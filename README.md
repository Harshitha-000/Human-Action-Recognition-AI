# 🧠 Human Action Recognition using Pose Estimation and Deep Learning

## 📌 Overview
This project is a real-time Human Action Recognition system that detects and classifies human activities using webcam input. It uses pose estimation to extract body landmarks and a deep learning model to recognize actions.

## 🚀 Features
- Real-time action detection using webcam
- Pose estimation using MediaPipe
- LSTM-based deep learning model
- Recognizes actions like walking, sitting, eating, drinking, and waving

## 🛠️ Technologies Used
- Python
- TensorFlow
- OpenCV
- MediaPipe
- NumPy
